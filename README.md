📊 Cricket Performance Analysis Dashboard (Power BI)
🔹 Project Overview

This project focuses on analyzing cricket player performance across different formats (ODI, T20, and Test) using Power BI.

The goal is to transform raw cricket statistics into meaningful insights through interactive visualizations.

🔹 Dataset

The dataset includes:

Player Name
Country
Matches, Innings
Runs, Average, Strike Rate
100s, 50s
Career Span (Start Year – End Year)

Data from multiple formats (ODI, T20, Test) were combined and cleaned.

🔹 Data Processing
Data cleaning using Power Query
Removed null values and unnecessary columns
Standardized column names and formats
Appended multiple datasets into a single table
🔹 Key Features
📌 KPI Metrics
Total Runs
Total Matches
Average Strike Rate
Average Batting Average
📌 Visualizations
Top 10 Players by Runs
Scatter Plot (Average vs Strike Rate)
Runs Comparison Across Formats
100s vs 50s Analysis (Player & Format level)
📌 Interactivity
Format Slicer (ODI, T20, Test)
Country Slicer
Year-based filtering
🔹 Tools & Technologies
Power BI
Power Query
DAX
🔹 Key Insights
Test format players generally accumulate higher total runs
T20 players show higher strike rates but lower averages
Top players maintain a balance between consistency and scoring speed
Performance varies significantly across formats and countries
🔹 Project Outcome

This project demonstrates:

Data cleaning and transformation
Data modeling and DAX usage
Dashboard design and storytelling
Interactive data visualization
